<p>FAQ:</p>
<ul>
<li>How to fetch data:
<ol>
<li>Create service inside <code>/src/services/modules/[modulesName]/index.tsx</code></li>
<li>Use RTK Query that already wrapped <code>/src/services/api.ts</code></li>
<li>Use build.mutation for DELETE, POST, PATCH, PUT</li>
<li>Use build.query for GET</li>
<li>Call wrapped mutation or query from page level like you call hooks</li>
<li><strong>You don't need to re-setup authorization token inside service module!</strong></li>
<li>Read the documentation https://redux-toolkit.js.org/rtk-query/overview</li>
</ol>
</li>
<li>What should I do if i need to store data
<ol>
<li><strong>Important! Don't store all data</strong></li>
<li><strong>Just store if you need to re-use the data from other modules OR multiple components OR you don't need to refetch</strong></li>
<li>Create store <code>/src/store/[modulesName]/index.tsx</code></li>
<li>Register store that you create inside combine reducer <code>/src/store/index.ts</code></li>
<li>Read the documentation https://redux-toolkit.js.org/api/createslice</li>
</ol>
</li>
</ul>
